📌 Project Overview

This project focuses on analyzing a Netflix movie dataset to explore movie trends, audience ratings, popularity patterns, and genre-based insights. The project involves data cleaning, exploratory data analysis (EDA), and data visualization using Python libraries to understand the characteristics of movies available on Netflix.

The main objective of this project is to extract meaningful insights from movie data and understand factors such as popularity, ratings, genres, release trends, and audience engagement.

🛠️ Technologies & Libraries Used

* **Python** – Programming language used for data analysis
* **Jupyter Notebook** – Development environment
* **Pandas** – Data cleaning, manipulation, and analysis
* **NumPy** – Numerical computations
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical data visualization

📂 Dataset Description

The dataset contains information about Netflix movies with the following attributes:

| Column Name       | Description                                         |
| ----------------- | --------------------------------------------------- |
| release_date      | Date when the movie was released                    |
| title             | Name of the movie                                   |
| overview          | Short description/storyline of the movie            |
| popularity        | Popularity score of the movie                       |
| vote_count        | Number of votes received by the movie               |
| vote_average      | Average rating given by users                       |
| original_language | Language in which the movie was originally released |
| genre             | Category or type of movie                           |
| poster_url        | URL of the movie poster                             |

🧹 Data Cleaning & Preprocessing

The following steps were performed to prepare the dataset:

* Checked dataset structure and information
* Handled missing values
* Removed duplicate records
* Checked and corrected data types
* Cleaned unnecessary or inconsistent data
* Prepared data for further analysis and visualization

📊 Exploratory Data Analysis (EDA)

Performed analysis to discover insights such as:

* Distribution of movie ratings
* Most popular movies based on popularity score
* Relationship between popularity and vote count
* Analysis of movie genres
* Release date trends
* Language-wise movie distribution
* Top-rated and most voted movies

📈 Data Visualization

Created different visualizations to understand patterns in the dataset:

* Bar charts for genre and language analysis
* Histograms for rating distribution
* Scatter plots to analyze relationships between variables
* Count plots for category comparison
* Other visualizations using Matplotlib and Seaborn

🔍 Key Insights

Through this analysis, the project helps in understanding:

* Which movies are more popular among users
* How ratings are distributed across movies
* Which genres have more content
* How movie releases changed over time
* The relationship between audience votes and movie popularity
